# public/catalogue - customer PDF downloads

This folder is regenerated by:

```powershell
npm run export-pdfs
```

The PDFs are exported from the refreshed catalogue files in
`final-deliverables/` using Microsoft Excel, so the owner's existing print
layout is preserved.

| PDF | Source workbook |
| --- | --- |
| `metals-catalogue.pdf` | `final-deliverables/Metals catalogue 2023.xlsx` |
| `mini-catalogue.pdf` | `final-deliverables/Mini Catalogue Self Updating.xlsm` |

These PDFs are tracked in git so Vercel can serve them on the website.
